# Clp

## ⛓️ Dependências para instalar

## 🚀 Como rodar?

```
# Instalar dependências
mix setup

# Rodar migrations
mix ecto.migrate

# Rodar seeds
mix run priv/repo/seeds.exs

# Rodar o servidor
mix phx.server

```

O servidor será inciado em [`localhost:4000`](http://localhost:4000).
